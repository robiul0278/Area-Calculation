ANSWER THE QUESTION

(1) What is DOM and what is the purpose of DOM?

Answer: DOM stands for Document Object Model. It is a programming interface that allows us to create, change, or remove elements from the document. We can also add events to these elements to make our page more dynamic. The DOM views an HTML document as a tree of nodes.



(2)) How will you select HTML elements using DOM? Name the DOM methods.

Answer: The word DOM stands for Document Object Model, where a web page is called a Document, and all tags inside the document are an object and the tree structure of every object the model.



(3)) What is event bubble?

Answer: Event bubbling in JavaScript can be defined as the method by which an event which got triggered within a nested set of HTML elements gets captured by the innermost element and travels up the DOM tree hierarchy to its outermost element



(4)) What is a callback function and why will you use it?

Answer: Callbacks are asynchronous functions which are executed when an event occurs. They are triggered automatically when some event occurs. When you setup a callback, you are just telling JS that call this function when some event occurs.
